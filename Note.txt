Quick Notes on Dependencies
Spring Web: Framework for creating web applications and REST APIs.
Spring Data MongoDB: Integrates MongoDB for NoSQL database functionality.
Lombok: Reduces repetitive code with annotations for common methods.
Spring Boot DevTools: Enhances development experience with features like live reload.
Spring Boot Actuator: Adds management and monitoring capabilities to your application.
JPAStreamer: Facilitates complex querying for JPA entities.
Build Helper Maven Plugin: Configures Maven to recognize additional source directories.
